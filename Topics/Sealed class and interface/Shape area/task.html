Shape area

You have a sealed class called Shape. Declare a function called calculateArea(), which takes the sealed class as a parameter and returns the area as a Double.

You can use the following formulas:

Square \(A = {a^2}\) where a is the side.

Rectangle \(A = {w l}\) where w is the width and l is the length.

Circle \(A = {πr^2}\) where r is the radius.

Triangle \(A = {hb_b/2}\) where b is the base and \(h_b\) is the height.

Pentagon \(A = {\frac{1}{4}{\sqrt{5(5+2\sqrt5)} }} \times a^2\) where a is the side.
